# Linux lab — home lab for IT support practice

Hands-on practice environment for Linux system administration and IT support skills.
Running on VirtualBox with Debian 13.

---

## Environment

```
Host OS:     Windows 10
Hypervisor:  VirtualBox 7.x
VMs:         1x Debian 13 server (headless)
Access:      SSH from host via Tabby
```

## What I practice here the next days 

| Area | Topics covered |
|------|---------------|
| SSH | Key-based auth, config hardening, troubleshooting |
| DNS / DHCP | Bind9 config, leases, dig/nslookup diagnostics |
| Users & permissions | useradd, groups, sudo, ACLs, passwd |
| Services | systemd units, enable/disable, journalctl |
| Logs | journalctl filters, /var/log, log rotation |
| Networking | iptables basics, ss, netstat, ping, traceroute |
| Scripting | Bash scripts for automation and diagnostics |
